\section{Conclusion}
\label{sec:conclusion}

\begin{BLACKSHEET}

This work adapts the Tsai-Lee framework for tree-based data structures to modern universal Software Transactional Memory Model by adopting descriptor-centric architecture. Our TLA+ analysis confirms that the proposed lightweight reading protocol—utilizing an Announce-Array and subscription-based barriers—maintains strict linearizability while significantly reducing helping overhead. This model demonstrates a successful balance between formal robustness and modern hardware constraints.
The source code for this project is available at: https://github.com/Yerk08/MCAS-formal-specification/

\end{BLACKSHEET}
